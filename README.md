## テスト駆動開発

### スタッククラスを作ろう!
テスト駆動開発で「スタック」クラスを作ってみよう。

### スタックとは

複数のデータを保持する。
- `push` で順に要素を追加できること。
- `pop` で最後から順に要素を取り出せること。

このような動作を LIFO (Last In, First Out) という。

### 実装するメソッド
- `empty?`
  - スタックが空なら真、空でなければ偽を返す。
- `size`
  - スタックの要素数を返す。
- `push(val)`
  - 引数の値をスタックの最後に積む。
- `pop`
  - スタックの最後の値を取り除いて返す。
  - スタックが空なら `Stack::EmptyStackError` を発生させる。
